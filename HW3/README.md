# Homework3

> Dataset: https://www.kaggle.com/datasets/abrambeyer/openintro-possum

> Что реализовано:
>
> * `DataReadWrite.scala` - модуль для чтения и записи данных в *csv*-формате.
> * `Scaler.scala` - модуль для стандартизации данных (вычитание среднего и деление на СКО).
> * `TrainTestSplit.scala` - модуль для разделения выборки на тренировочную и валидационную части.
> * `LinearRegression.scala` - модуль линейной регрессии, реализованы аналитический вариант (*fitAnalytical*) и SGD (*fitSGD*).
> * `Metrics.scala` - модуль для расчета метрик, реализованы *R2* и *RMSE*.
> * `Main.scala` - основной модуль, где реализован пайплайн обучения и валидации.

> В директории *Data* находятся исходные файлы с тренировочными и тестовыми данными. Сюда же происходит сохранение результатов прогноза для валидационных и тестовых данных.
